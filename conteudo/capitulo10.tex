\chapter*{Apêndice D - Projetos}\label{projeto}

O desenvolvimento do Portal de Algoritmos foi dividido em dois códigos bases, \textit{backend} e \textit{frontend} e foi feito o versionamento de código utilizando o protocolo \textit{git} e repositório de código \textit{bitbucket}.

\section*{Backend}

O \textit{Backend} foi desenvolvido em Java.

Para começar a desenvolver é preciso clonar o projeto do \textit{bitbucket} e importar o mesmo na sua \ac{IDE} de preferência. As dependências do projeto são mantidas no arquivo pom.xml e gerenciadas pelo \textit{maven}, então basta \textit{buildar} o projeto com ele.

O servidor de aplicação utilizado foi o \textit{Wildfly}, é preciso então baixar e configurar localmente ou no servidor de aplicação.

Link do projeto: \url{https://bitbucket.org/adrianomargarin/portalg}

\section*{Frontend}

O \textit{Frontend} foi desenvolvido com \textit{Angular 4}, \ac{HTML} e \ac{CSS}. As dependências do projeto são mantidas no arquivo package.json e para instalar é utilizado o comando a seguir.

\begin{lstlisting}
npm install
\end{lstlisting}

Para executar o projeto é preciso levantar o servidor de aplicação com o comando a seguir.

\begin{lstlisting}
ng serve
\end{lstlisting}

Link do projeto: \url{https://bitbucket.org/adrianomargarin/portalg-frontend}
